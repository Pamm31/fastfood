## How to run locally

\_
1. Clone this repo using:
```bash
git clone https://github.com/Pamm31/fastfood.git
```
or if you have a SSH token:
```bash
git clone git@github.com:Pamm31/fastfood.git
```
\_

2. Run:
```bash
yarn install
```
or
```bash
npm install
```
\_

3. Run 
```bash
yarn dev
```
or
```bash
npm run dev
```
